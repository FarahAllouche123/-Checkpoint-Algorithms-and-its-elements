Analyse de Phrase
#Description

Ce projet contient un algorithme qui analyse une phrase saisie par l’utilisateur et calcule :

La longueur de la phrase (jusqu’au point .)

Le nombre de mots

Le nombre de voyelles

L’algorithme utilise uniquement 3 compteurs (len, words, vowels) comme demandé pour le projet.

#Fonctionnement

1-L’utilisateur saisit une phrase se terminant par un point ..

2-Le programme parcourt chaque caractère de la phrase :

    1-Compte la longueur jusqu’au point.

    2-Incrémente le compteur de mots à chaque espace.

    3-Incrémente le compteur de voyelles si le caractère est a, e, i, o, u (majuscules ou minuscules).

3-Affiche les résultats à l’écran.
